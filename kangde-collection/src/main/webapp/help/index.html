新手指南

康得金控风控系统V1.0主要用于康得金控资产管理有限公司内部的风控业务，产品在权限控制、数据管理等方面进行了重点改进，下面为系统的简单操作流程。

操作流程：
简要描述：

一、登录

用户名为原系统用户名，初始密码为1234abcd，建议用户登录后在用户中心进行修改.

二、系统管理

1、组织机构

系统管理员有权限新增修改组织机构（包括总部、分公司、小组、客服、外访等机构）

2、员工管理

系统管理员有权限新增修改员工，其中本系统针对本公司业务定制化了附加机构、关联角色、和委托方的字段。

1)附加机构：附加机构指的是用户所涉及到的业务区域，现有的权限管理制度决定用户只能看见自己所属区域的案件，因此，在新增用户的时候，当一个用户负责多个区域的业务的时候，可以通过附加区域的设置来解决此问题。

2)关联角色：关联角色决定了用户的权限的设置，如客服、风控员、风控主任、风控经理、外访主任等角色。

3)委托方：委托方主要针对客服角色，由于客服的工作中会是有客服对应委托方的原则存在，所以，当委托方字段为空时，可见案件队列根据所属机构和附加机构去判断；当选择了相应的委托方后，案件队列就会以该委托方为第一优先级，只能看见该委托方所有机构下面的案件。

3、委托方管理

客服人员有权限在委托方管理中新增修改委托方信息，目前为基础的信息整理功能，在接下来的版本中会继续丰富该模块。

4、模板配置

1)外访信函模板 信函模板和外访模板采用同种形式，在创建信函时需要选择模板类型，然后将右侧对应的代码粘贴到编辑器中即可制作模板。

2)案件模板 案件模板是针对客服在案件管理中进行案件导出时需要的模板，可以针对不同的委托方进行不同的模板配置，这样在以后导出时可直接选择模板进行导出。

3)催记模板 催记模板是是针对客服在催记管理中进行催记导出时需要的模板，可以针对不同的委托方进行不同的模板配置，这样在以后导出时可直接选择模板进行导出。

三、案件管理

案件管理模块主要针对客服角色。

1、批次管理

批次管理中主要包括新增、编辑、删除、导出、更新几个功能。

1)新增

新增时，按列表逐个填写内容，模板可通过底下的链接下载，其中批次号为自动生成且不可修改。

2)导出-更新

导出和更新是相对的功能，更新前，需要导出该案件，选择需要更新的字段进行导出，导出后，在excel中进行修改，修改后，点更新批次，上传文件，即可。

2、案件管理

案件管理中主要包括暂停、撤回、恢复、结清和导出案件。

1)暂停

暂停功能是变更案件的状态，暂停时风控员还可在我的案件中看见该案件。

2)撤回

撤回功能是将案件从风控员那撤回，风控员在我的案件中不能看见该案件。

3)结清

结清功能是对案件的手动结清，点击结清后，不管案件的待还金额是否为“0”，案件状态都将变为结清。

4)恢复

恢复功能是对暂停、撤回、结清的案件进行恢复，恢复后案件状态为正常。

5)导出

点击导出后，系统会弹出弹窗，在筛选框中选择相应模板，即可进行导出。

3、留案管理

留案管理是客服对风控员在案件详情页进行留案申请后的审批页面，审批通过后，案件的退案日期会自动顺延至选择日期。

4、还款管理

还款管理模块，客服对风控录入CP后与委托方进行还款后，可在该模块确认还款。

1)确定还款-作废还款

在与委托方对账后，客服可以点击确定还款，输入确认金额，当还款金额大于等于剩余还款金额时，系统会提示是否结清案件，选择结清，则案件的状态将变为结清。

2)导出

在客服与委托方去对账时，可选择相应字段，如档案号、案件批次号等进行导出，导出成excel文件。

5、信函管理

信函管理为信函邮寄的记录模块，流程也是由风控员在案件详情页发起，由客服去审批邮寄。

1)待审批信函

该模块中的数据为风控员在案件详情中进行的申请，客服可以进行审批，选择通过或者不通过。

2)待处理信函

该模块中的数据为审核通过，等待邮寄的申请。客服可以点击导出信函，选择信函模板，到处成word文件，然后通过快递等形式进行寄出，寄出后选择邮寄信函，这样信函的状态会变成已完成。

3)信函记录

信函记录中的数据是已经寄出的信函，当有信函被退回后，查找到对应信函，点击退回按钮，这样信函的状态会变为被退回。

四、风控管理

1、分案管理

分案管理是对操作者所属公司的案件进行分案，非本公司案件在本模块看不见。

1)所选分案

对所选的案件进行分案，只能指定一个风控员。

2)所查结果分案

对通过筛选条件查出的所有案件进行分案，只能指定一个风控员。

3)所查自动分案

对通过筛选条件查出的所有案件进行分案，可以选择是否包含已分配的案件，若包含已分配的案件，则这些案件将会被分配给新的风控员。在分配方式上可以选择按数量分配，或是按金额分配，然后选择风控员们，输入百分比，即可实现自动分案

4)所选自动分案

与所查自动分案操作相同，不过针对的案件是在列表中选择的案件。

2、风控管理

1)我的案件

该列表页为所有分配给操作者的案件，点击案件序列号即可进入案件详情页。

在案件信息部分展示的是贴案过来的信息，可以点击左侧按钮展开隐藏的内容。

在操作台部分为风控员日常催收的部分，包括电话、地址、风控记录、外访记录、登账记录、资料修复、共债案件、留案记录、协催记录、信函记录、评语部分。

A.电话

先登录天润，登录后点击电话后面的加号可以直接拨打电话。拨完电话后，刚刚拨打的电话信息会自动带入底下的催记模块，这时风控员需要及时填写催记，当拨打下一个电话时，无法补记催记。

B.地址

地址模块涉及到外访和信函的申请，风控员需要先将地址维护成省/市-市-县/区-详细地址的模式，然后才能申请，且每个地址在外访（信函）完成之前只能申请一次。申请后，申请会进入到待审批外访/信函模块，由相关负责人进行审批，风控员可在外访记录/信函记录模块看见进度。

C.风控记录

风控记录是所有风控员对该案件的风控记录。

D.外访记录

风控员可在该模块看见已申请待审批、待处理、和已完成的外访记录。当分配外访员后，风控员可查看外访员联系方式，便于沟通；外访完成后，风控员可在附件中查看外访员上传的附件。

E.登账记录

登账记录中包含风控员在录催记时录入的PTP记录和直接录入的CP记录，在录入CP时，可上传凭证。录入后，客服可在还款管理模块看见该条记录。

F.资料修复

资料修复指对现有数据的修复，包括户籍资料（HJ）、社保资料（SHB）、电信资料（DX），点击申请后，选择相应类别，填写申请内容，申请后，可看见记录，当查资完成后，可在该模块看见查回的资料，并将查回的资料点击“+”添加至电话/地址簿中。

G.共债案件

在共债案件中可以看见同身份证号债务人的不同案件，点击案件序列号可以看见案件详情。

H.留案记录

在留案记录中可看见该案件的留案历史，申请留案后，当客服审批通过后，案件即可按客服确认的时间顺延到期日。

I.协催记录

协催内容包括客服咨询、公安协助、法院协助、主管协催，申请后由客服进行回复。

J.信函记录

信函记录为对之前由地址栏进行申请的信函的记录。

K.评语

评语为对该案件的点评，客服、风控员、风控主任、经理均可进行点评。

2)部门案件

部门案件为风控主任、风控经理、和分公司负责人所查看，旨在查看其所负责的风控员的案件情况。

3、外访管理

1)待审批外访

风控员在地址簿中申请后，外访主任或风控经理即可在该模块审批，通过后即进入排程模块。

2)待排程外访

外访主任或风控经理可对外访进行排程，选择相应外访员；此外，在该阶段，还可直接撤销外访，撤销后，案件回到待审批的列表中。

3)已排程外访

已排程外访为分给操作者的外访案件。

A.导出所选外访单

批量选择外访单，点击导出，选择需要的外访模板进行导出，导出后为word格式。

B.完成外访

在外访完成后，需要对每个外访提交外访报告，外访结果和时间，上传外访附件，上传完成后，外访员无法进行修改和删除。

C.返回排程

返回排程后的案件即重新让外访主任进行排程。

D.撤销外访

撤销外访会返回到审批列表。

4)已完成外访

该列表中的案件为已经完成的外访的汇总，在导出外访内容时，选择需要的字段进行导出即可。

4、查资管理

1)待审批查资

该模块指风控员的户籍、社保、电信的查资申请列表，风控经理在该模块进行审批。

2)待处理查资

审核通过后的申请会进入到该列表中，风控主任可以通过查资类型进行筛选，如“HJ”类，筛选后导出查资记录，查询完成后，需要将资料录入到导出的excel中，保存后，点击导入查资记录，选择相应文件，即可。若单个查询，也可在列表中的操作栏进行编辑，编辑后保存即可。

3)查资记录

在该模块可查询历史的查资记录，可通过风控方、申请日期、案件序列号、查资类型、申请人、债务人姓名、证件号进行查询。

5、协催管理

1)待处理协催

待处理协催包括风控员提交的客服咨询、公安协助、法院协助、主管谢翠等的内容，风控主任、风控经理以及客服可在该模块进行回复。

2)协催记录

协催记录为历史协催的汇总记录，可通过案件序列号、风控方、协催类型进行查询。

五、催记管理

客服可在催记管理模块查看、导出想要的案件的催记。导出时，选择设置好的催记模板即可导出成excel文件。
